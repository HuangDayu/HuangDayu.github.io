## Java静态代码块

---

```java
public class Test{
    Map<String, String> map = new LinkedHashMap<String, String>();
    static{
        map.put("Test", "static");
    }
}
```
* 类被加载后，只执行一次
* 可用户数据初始化，Map，List等

## Java动态代码块

---

```java
public class Test{
    Map<String, String> map = new LinkedHashMap<String, String>();
    {
        map.put("Test", "static");
    }
}
```

* 对象被创建一次（`new()`） 就执行一次
* 无条件加载在构造方法的前面，无论构造方法有无参数
