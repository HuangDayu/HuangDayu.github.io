远程重启tomcat  

```shell
sudo apt install sshpass
```

restart.sh  

```shell
#!/bin/bash
sshpass -p $1 ssh root@ali.huangdayu.cn "/usr/local/src/tomcat/tomcat-1/bin/shutdown.sh && /usr/local/src/tomcat/tomcat-1/bin/startup.sh"
```

执行时在文件后面跟随服务器密码  

```shell
sh restart.sh servicePassword
```

说明：  

**`$1`** 用于获取文件名后面跟随的参数
